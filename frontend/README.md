# Deadline Guardian AI - Frontend

This is the frontend React client for the Deadline Guardian AI application.

For full project documentation (including architecture diagrams, database schema, API documentation, and setup instructions), please refer to the **[Root README.md](../README.md)**.

## Developer Quick Start

To run the frontend in development mode on its own (assuming backend is running elsewhere):
```bash
npm install
npm run dev
```

To build the static production bundle:
```bash
npm run build
```
The compiled build will be generated in the `dist` directory, which can be served by the backend or any web server.

# Deadline Guardian AI - Backend

This is the backend REST API server for the Deadline Guardian AI application.

For full project documentation (including architecture diagrams, database schema, API documentation, and setup instructions), please refer to the **[Root README.md](../README.md)**.

## Developer Quick Start

To run the backend server in isolation:
```bash
npm install
npm start
```

To run the automated tests:
```bash
npm test
```

To run the tests with a coverage report:
```bash
npm run test:coverage
```
Ensure that you have created and configured a `.env` file in this directory based on the `.env.example` file, and placed your `firebase-service-account.json` in this directory.
